Set 
====
Duplicate elements are not allowed


Implementation
===============
HashSet
LinkedHashSet
TreeMap

HashSet--> is implemented by HashMap it allows inserting of null
	   --> With HashSet order of the element is not maintained	
LinkedHashSet --> Maintains the insertion order of the element
			  --> implemented by Hash table and linked list	
			  --> Allow insertion of null	   
TreeSet-->is implemented by TreeMap it allows inserting of null
	   -->With Tree set natural ordering of element is maintained
	   -->TreeMap is implemented as RED-BLACK Tree
	   	



No sort method 
No get method 
No add method to specific position
